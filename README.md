# webportfolio
Personal Portfolio Website with Admin Dashboard

## Features
- Minimalist dark theme portfolio
- Admin dashboard for content management
- Sections: Hero, About, Projects, Experience, Education
- localStorage-based CMS
- Export/Import data as JSON

## How to Use
1. Open `index.html` for the portfolio
2. Open `admin.html` for the admin dashboard (password: `admin123`)

## Deployment
Hosted on GitHub Pages
